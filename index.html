Estrutura básica do HTML

O HTML possui uma estrutura básica que consiste em:

<html>
<head></head>
<body></body>
</html>


A tag <html> é o elemento raiz de uma página HTML

A tag <head> é a cabeça do html, lá fica armazenado o que é preciso ser iniciado antes da página ser carregada.

A tag <body> é o corpo do html, lá vai estar todo o conteúdo da página.

O que são TAGS?

As TAGS informam para o navegador o tipo de estrutura que está sendo construída, podendo ser titulos, paragrafos, links, imagens e muitos outros.

Falando sobre Atributos

O que são atributos?

Atributos são palavras especiais utilizadas dentro das tags de abertura para controlar o comportamento do elemento. Ha dois tipos de atributos em HTML que são:


Atributos globais

Os atributos globais são aceitos por todas as tags como por exemplo: class, id, lang, style e etc.


Atributos específicos

Esses atributos possuem tags especificas que podem ser adicionados, são exemplos de atributos específicos: src, disabled, href, label e etc.
